# Swagger UI

Swagger UI is enabled with `springdoc-openapi-starter-webmvc-ui`.

After running the services:

- User API docs: `http://localhost:8081/swagger-ui/index.html`
- Order API docs: `http://localhost:8082/swagger-ui/index.html`

Swagger helps inspect endpoints, request bodies, response shapes, and validation rules without needing a separate API client.
